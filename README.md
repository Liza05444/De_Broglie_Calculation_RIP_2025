# Разработка интернет-приложений (РИП) 2025

## Расчет длины волны де Бройля для частиц

Разработанное приложение позволит исследователям создавать и формировать заявки на расчёты длины волны де Бройля для выбранных ими частиц, а профессорам завершать или отклонять данные заявки.
